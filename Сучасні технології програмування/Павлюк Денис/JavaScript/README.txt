Тема: Розробка інтерактивної симуляції гравітаційної взаємодії N тіл засобами JavaScript.
1. Опис роботи програми: Програма являє собою веб-додаток, що моделює фізичну взаємодію небесних тіл у двовимірному просторі. Користувач має можливість створювати нові об'єкти ("планети") за допомогою миші або сенсорного екрану, задаючи їм початкову позицію та вектор швидкості (принцип "рогатки").
2. Технічна реалізація: Проект написаний чистою мовою JavaScript (Vanilla JS) без використання сторонніх бібліотек.
Основні технічні аспекти:
--HTML5 Canvas API: Використовується для графічного виведення.
--Об'єктно-орієнтоване програмування (ООП): Основною структурною одиницею є клас Planet. Він інкапсулює властивості об'єкта (координати x, y, швидкість vx, vy, масу mass) та його поведінку (методи draw() для малювання та update() для розрахунків).
--Цикл анімації: Реалізовано через метод requestAnimationFrame().
--Обробка подій: Використовуються слухачі подій (addEventListener) для відстеження дій миші та дотиків, що робить інтерфейс інтерактивним.
3. Алгоритм та фізика: У методі update() реалізовано спрощену модель закону всесвітнього тяжіння Ньютона. Для кожної пари планет розраховується:
--Вектор відстані між центрами.
--Сила тяжіння.
--Зміна швидкості, що додається до поточного вектора руху об'єкта.

(Якщо коротко, то можна весело стріляти різнокольоровими планетами.)
