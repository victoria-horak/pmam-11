# ChatSimulator - Python

## Опис

(GUI) Симулятор чату з підтримкою кількох користувачів, який дозволяє надсилати повідомлення, зберігати історію і перемикати теми. Реалізований на **Python** з використанням бібліотеки **Tkinter** для графічного інтерфейсу.

## Ключові Файли

| Файл | Призначення |
| :--- | :--- |
| `main.py` | Точка входу. |
| `app/chat_app.py` | Контролер. Керує взаємодією між вікном (ChatWindow) і менеджером повідомлень (MessageManager). |
| `app/chat_window.py` | GUI-компонент (Tkinter-вікно). Відповідає за відображення, введення даних, теми і взаємодію з користувачем. | 
| `app/message_manager.py` | Менеджер даних. Керує додаванням, отриманням, очищенням повідомлень і забезпечує збереження/завантаження історії в `chat_history.json`. |
