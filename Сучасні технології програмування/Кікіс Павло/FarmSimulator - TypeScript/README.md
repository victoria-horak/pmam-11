# FarmSimulator - TypeScript

## Опис

(CLI) Текстовий симулятор ферми, що дозволяє користувачеві керувати рослинами та тваринами. Гравець може садити культури, збирати врожай, годувати і грати з тваринами, а також стежити за балансом ферми. Реалізований на **TypeScript** і виконується за допомогою **Node.js** з використанням вбудованого модуля `readline` для взаємодії з користувачем.

## Ключові Файли

| Файл | Призначення |
| :--- | :--- |
| `main.ts` | Містить всю логіку гри, включаючи класи **Plant**, **Animal** і головний клас **Farm**, а також основний цикл гри `gameLoop`. |
