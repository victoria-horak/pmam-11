# OrderQueue - GO

## Опис

(CLI) Додаток для управління чергою замовлень із підтримкою пріоритетів. Система дозволяє додавати, обробляти (прибирати з черги), видаляти, редагувати і шукати замовлення. Черга автоматично сортується за **пріоритетом** (1-3), де **1** — найвищий. Реалізований на **Go (Golang)**, використовує стандартні бібліотеки для JSON-серіалізації і зберігання даних.

## Ключові Файли

| Файл / Папка | Призначення |
| :--- | :--- |
| `main.go` | Точка входу. Керує CLI-інтерфейсом, приймає команди користувача і викликає методи з пакета `services`. |
| `services/order_queue.go` | Головний клас **OrderQueue**. Містить усю бізнес-логіку черги: додавання, сортування, вилучення (`Pop`), пошук, редагування і розрахунок статистики. |
| `models/order.go` | Структура даних **Order**. Визначає поля, необхідні для замовлення (ID, CustomerName, Item, Priority, CreatedAt). |
| `storage/storage.go` | Логіка зберігання даних. Відповідає за **завантаження** і **збереження** черги замовлень у файл `data/orders.json`. |
| `data/` | Директорія, яка автоматично створюється для зберігання файлу даних `orders.json`. |
