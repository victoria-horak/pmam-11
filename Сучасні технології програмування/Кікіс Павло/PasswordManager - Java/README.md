# PasswordManager - Java

## Опис

(GUI) Менеджер паролів, реалізований на **Java** з використанням бібліотеки **Swing**. Додаток дозволяє користувачеві зберігати, редагувати, видаляти і шукати облікові дані. Зберігання всіх даних у зашифрованому вигляді у локальному файлі (`passwords.dat`) за допомогою простого алгоритму **XOR-шифрування**, поєднаного з Base64-кодуванням.

## Ключові Файли

| Файл | Призначення |
| :--- | :--- |
| `src/Main.java` | Головний клас програми і точка входу. Створює головне вікно (`JFrame`), налаштовує GUI (темна тема, таблиця, кнопки, пошук) і керує взаємодією з користувачем. |
| `src/Storage.java` | Відповідає за **збереження** і **завантаження** даних з файлу `passwords.dat`. Використовує клас `Crypto` для шифрування/дешифрування вмісту файлу. |
| `src/Crypto.java` | Клас, що реалізує логіку **шифрування** і **дешифрування** даних. Використовує просту операцію XOR із статичним ключем і Base64-кодування. |
| `src/PasswordEntry.java` | Модель даних. Визначає структуру об'єкта для зберігання одного облікового запису (title, username, password, url, notes, updated). |
| `src/PasswordGenerator.java` | Клас для генерації надійних випадкових паролів на запит користувача. |
| `passwords.dat` | Файл даних, у якому зберігаються **зашифровані** облікові записи. |
