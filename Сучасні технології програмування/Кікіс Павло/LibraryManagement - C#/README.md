# LibraryManagement - C#

## Опис

(GUI) Додаток для управління особистою бібліотекою і обліком користувачів. Реалізований з **Windows Forms** на **C#**. Він підтримує реєстрацію/вхід користувачів (з хешуванням паролів за допомогою BCrypt), перегляд загальної бібліотеки, додавання книг до особистої колекції і пошук.

## Ключові Файли

| Файл | Призначення |
| :--- | :--- |
| `Program.cs` | Точка входу. Ініціалізує графічний інтерфейс і запускає першу форму (`SigninForm`). |
| `MainForm.cs` | Головна форма. Містить логіку відображення бібліотеки, управління книгами користувача, пошук і розрахунок рейтингів. |
| `SigninForm.cs` | Форма входу. Відповідає за аутентифікацію користувачів шляхом порівняння введених даних із даними у `users.txt`. |
| `SignupForm.cs` | Форма реєстрації. Керує реєстрацією нових користувачів, перевіряє надійність паролів і зберігає хешовані дані. |
| `Book.cs` | Модель даних (`struct`), описує структуру книги (назва, автор, жанр, рік). |
