# ExpenseTracker - Ruby

## Опис

(CLI) Додаток для керування і відстеження особистих фінансових витрат. Він дозволяє користувачам додавати, видаляти, переглядати, шукати та фільтрувати витрати, а також отримувати детальну статистику за категоріями і місяцями. Реалізований на **Ruby** і використовує JSON для збереження даних.

## Ключові Файли

| Файл / Папка | Призначення |
| :--- | :--- |
| `main.rb` | Точка входу. Відповідає за виведення головного меню і обробку команд користувача (CLI). |
| `lib/expense_tracker.rb` | Головний клас **ExpenseTracker**. Містить всю логіку керування витратами: додавання, видалення, завантаження/збереження, розрахунок статистики і експорт у CSV. |
| `lib/expense.rb` | Клас **Expense**. Модель даних, яка інкапсулює інформацію про одну витрату. |
| `storage/` | Директорія, де зберігається файл `expenses.json` з усіма записами витрат. |
