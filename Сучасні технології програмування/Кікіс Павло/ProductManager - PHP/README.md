# ProductManager - PHP

## Опис

(CLI) Додаток для управління інвентаризацією складу. Дозволяє користувачам додавати, оновлювати, видаляти товари, виконувати пошук та переглядати статистику (загальна вартість, категорії). Підтримка **відкату (Rollback)** останньої дії, використовуючи лог дій. Реалізований на **PHP** з використанням локальних JSON-файлів для зберігання даних.

## Ключові Файли

| Файл | Призначення |
| :--- | :--- |
| `main.php` | Точка входу. Керує CLI-інтерфейсом, відображає меню і обробляє команди користувача. |
| `src/ProductStorage.php` | Клас, що містить усю логіку управління даними: завантаження/збереження, додавання, оновлення, видалення, пошук, статистика і механізм **відкату (Rollback)**. |
| `src/Product.php` | Клас-модель **Product**. Визначає структуру товару (id, name, price, quantity, category, createdAt). |
| `storage/` | Директорія для зберігання даних. Містить файл `products.json` (список товарів) і `log.txt` (історія дій). |
